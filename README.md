# LogseqHelper
![](LogseqHelper.png?raw=true)  
LogseqHelper is an organizer application that allows Logseq users to have more overview over their Logseq directory.

# Features
 - Look at stats of your Logseq directory
 - View your images and videos and find out where they are linked by hovering over them
 - Find unlinked pages and link them
 - Export your Logseq directory graph as a GEXF file for visualization and analysis in tools like [Gephi](https://gephi.org/)

# Usage
1. Configure the application under "src/config.py"
2. Build the application using "build.sh" or "build.bat"
3. Place the application next to your Logseq directory (default behaviour in "src/config.py")

# Note
This was tested on a Linux machine.
